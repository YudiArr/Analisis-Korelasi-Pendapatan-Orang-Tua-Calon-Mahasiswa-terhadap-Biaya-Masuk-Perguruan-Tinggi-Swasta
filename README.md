# Analisis-Korelasi-Pendapatan-Orang-Tua-Calon-Mahasiswa-terhadap-Biaya-Masuk-Perguruan-Tinggi-Swasta

 
Konsultan dari sebuah perguruan tinggi swasta melakukan evaluasi terkait kinerja proses penerimaan mahasiswa baru selama sepuluh tahun terakhir, mulai dari 2015 s.d. 2025 baik pada semester gasal dan genap. Data menunjukkan bahwa jumlah mahasiswa baru dalam kurun waktu tersebut adalah cenderung turun. Dari hasil investigasi sementara, menunjukkan bahwa pendapatan orang tua adalah turunnya jumlah mahasiswa baru. Pimpinan kampus ingin melihat korelasi antara pendapatan orang tua calon mahasiswa dan biaya masuk perguruan tinggi.
